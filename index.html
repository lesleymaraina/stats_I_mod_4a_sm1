Transition from Statistical Foundations to Hypothesis Testing

  • hypothesis testing builds on the concepts developed in earlier lectures by integrating sample summaries, probability models, sampling variability, standard errors, and confidence intervals into a formal inferential procedure

As a review:

  • estimation procedures yield point estimates and corresponding standard errors for population parameters
  • confidence intervals quantify statistical uncertainty and define ranges of parameter values supported by the observed data
  • hypothesis testing uses these components to formally evaluate whether the observed data are compatible with a null hypothesis regarding a population parameter

Approaches allow researchers to determine whether an observed treatment effect, biomarker association, or group difference is larger than would reasonably be expected from random sampling variability alone under a null hypothesis of no effect or no association

Why This Matters in Biomedical Data Science

hypothesis tests provide a way to compare observations with what would be expected under a null assumption

Analysts are often asked to evaluate whether a signal is statistically meaningful

Examples include:

  • whether a biomarker is associated with tumor response
  • whether a patient subgroup has a different response probability
  • whether tumor burden changes after therapy
  • whether a clinical feature is associated with progression risk
  • whether a model term improves explanation of an outcome

Hypothesis Test : Definition

hypothesis test is defined as:

a method that uses sample data to decide between two competing claims [hypotheses] about a population characteristic

Where:

The null hypothesis represents the default population claim and the alternative hypothesis represents the competing scientific claim evaluated using the data

Null hypothesis \(H_0\)

  • claim about a population characteristic that is initially assumed to be true
  • serves as the reference point for evaluating evidence

Alternative hypothesis \(H_A\)

  • competing claim about the same population characteristic
  • represents the type of difference or association the study is designed to detect

Hypothesis Testing: A Stepwise Approach

Statement of the Problem

  1. Describe the population characteristic of interest
  2. State the null hypothesis, \(H_0\)
  3. State the alternative hypothesis, \(H_A\)
  4. Select the significance level, \(\alpha\)

Analysis

  1. Display the test statistic, with the hypothesized value from \(H_0\) substituted into the formula
  2. Check that required assumptions are reasonable
  3. Calculate all quantities in the test statistic and calculate the test statistic itself
  4. Determine the p-value associated with the observed test statistic

Conclusion

  1. State the conclusion: reject \(H_0\) if p-value \(\leq \alpha\), and fail to reject \(H_0\) otherwise. Then state the conclusion in the context of the problem, including the significance level.

Hypothesis Testing: An Oncology Example

Hypothesis testing evaluates whether observed sample evidence supports or contradicts a proposed explanation about a population

Primary question:

Is the probability of tumor response to therapy higher among patients with a targeted biomarker than among patients without the biomarker?

Describe the Population Characteristic of Interest

Step 1: Describe the population characteristic

population characteristic: unknown population-level quantity the hypothesis test is designed to evaluate

Example:

\[ p=\text{population probability of tumor reduction} \]

State the Null and Alternative Hypothesis

Step 2: State the null hypothesis \(H_0\)

null hypothesis \(H_0\): claim about a population characteristic that is initially assumed to be true

  • provides the baseline explanation for the observed data
  • serves as the reference model against which the observed data are evaluated

\[ H_0 : p = 0.5 \]

  • probability of tumor response equals 50%

State the Null and Alternative Hypothesis

Step 3: State the alternative hypothesis \(H_A\)

alternative hypothesis \(H_A\): competing claim suggesting the population characteristic differs from the null hypothesis

different forms of the alternative hypothesis reflect different scientific questions about the population parameter and determine whether statistical evidence is evaluated in one direction or in both directions relative to the null hypothesis value

Two-Sided Alternative Hypothesis

\[ H_A:p \neq 0.50 \]

  • Asks: Is the population response probability different from 50%?

  • asks whether the true response probability is different from 50% in either direction

  • both higher and lower response probabilities count as evidence against H_0

One-Sided Alternative Hypothesis

\[ H_A:p>0.50 \]

  • Asks: Does the biomarker improve response probability?

  • asks whether the response probability is greater than 50%

  • only increases count as evidence against \(H_0\)

  • Essentially: if the observed response probability were actually lower than 50% this would not support this alternative hypothesis

Significance levels and Type I/II Errors

4. Select Significance Level \(\alpha\)

significance level \(alpha\) : probability threshold used to define evidence strong enough to reject the null hypothesis

  • smaller α → stricter evidence required
  • larger α → easier to reject \(H_0\)

Hypothesis testing decisions can produce two major categories of statistical error:

  • rejecting \(H_0\) when it is actually true
  • failing to reject \(H_0\) when it is actually false

Type I Error: error of rejecting \(H_0\) when \(H_0\) is true [\(\leftarrow\) significance levels controls this error probability]

  • \(\alpha\): probability of a Type I error

e.x.: concluding that a biomarker is associated with response when it is not

Type II Error: failing to reject the null hypothesis [\(H_0\)] when the null hypothesis [\(H_0\)] is actually false

  • \(\beta\) : probability of a Type II error

e.x.: failing to detect a biologically meaningful signal

Interpreting the Significance Level

Choosing \(\alpha\) reflects how much risk of a Type I error is acceptable given the scientific question

Common thresholds for \(\alpha\):

  • α=0.05
  • α=0.01

If \(\alpha = 0.05\):

  • reject \(H_0\) if p-value < 0.05 : statistical significance
  • in repeated testing when \(H_0\) is true about 5% of tests will incorrectly reject \(H_0\)
  • 5% probability of falsely concluding that an effect exists when the null hypothesis is true

If \(\alpha = 0.01\):

  • reject \(H_0\) if p-value < 0.01 : statistical significance
  • in repeated testing when \(H_0\) is true about 1% of tests will incorrectly reject \(H_0\)
  • 1% probability of falsely concluding that an effect exists when the null hypothesis is true
  • fewer false positives are expected

Display the Test Statistic

5. Display test statistic that will be used in analysis

test statistic : numerical quantity calculated from sample data used to decide whether to reject or fail to reject \(H_0\)

Since the characteristic of interest is a population proportion:

\[ p = \text{population probability of tumor response} \]

  • parameter being tested is a proportion
  • one-sample z statistic measure comparing an observed sample proportion to the null hypothesis proportion \(H_0\)

Display the Test Statistic

  • different test statistics can be used for hypothesis tests
  • each statistic quantifies how much the observed sample varies from the null hypothesis

Many hypothesis tests use a statistic with this structure:

\[ \text{test statistic} = \frac{\text{estimate} - \text{null value}}{\text{standard error}} \]

Where:

  • estimate: sample-based estimate of the population characteristic
  • null value: value stated in \(H_0\)
  • standard error: estimated variability of the estimator across repeated samples

Display the Test Statistic

Choice of test statistic depends on:

  • type of outcome variable
  • population characteristic being tested
  • statistical model used for the data

Examples include:

Test statistic Use Description
z statistic population proportions and some large-sample mean tests
t statistic population means with estimated variability
chi-square statistic \(\chi^2\) categorical data and contingency tables
F statistic comparing variability across groups in ANOVA

Display the Test Statistic

5. Display test statistic that will be used in analysis

  • one-sample z statistic measures how far the observed response proportion differs from the null response probability after accounting for expected sampling variability

\[ z = \frac{\text{estimate [observed value] - expected [null]}}{\text{standard error}} = \frac{ \hat{p} - p_0 }{ \sqrt{ \frac{ p_0(1-p_0) }{n} } } \]

Where:

  • \(\hat{p}\): observed tumor response proportion in the sample
  • \(p_0\): hypothesized response probability under the null hypothesis
  • \(n\): sample size
  • denominator: standard error expected if \(H_0\) were true

Display the Test Statistic

5. Display test statistic that will be used in analysis

Hypothesis:

determine whether the population tumor response probability is greater than 50%

As a result:

  • [large] positive z statistics support \(H_A\)
  • [large] negative z statistics do not support \(H_A\)

Interpretation:

  • z statistic near 0 → observed response proportion is consistent with \(H_0\)
  • large positive z → observed response proportion is much larger than expected under \(H_0\)
  • large negative z → observed response proportion is much smaller than expected under \(H_0\)

Suppose you have a dataset of 120 cancer patients

The variable response is binary:

1 = tumor response observed

0 = tumor response not observed

Observed response proportion \(\hat{p}\) estimates the population response probability p

\(\hat{p}\): observed sample proportion [averge across all responses in dataset]

\(p_0\): null hypothesis proportion [p = 50%]

Hypothesis Test for Tumor Response Probability

p0 <- 0.50

summary_df <- tumor_response_df |>
  summarise(
    n = n(),
    p_hat = mean(response)
  ) |>
  mutate(
    se_null = sqrt(p0 * (1 - p0) / n),
    z = (p_hat - p0) / se_null,
    p_value = 1 - pnorm(z)
  )

summary_df

Code Explanation

p0 <- 0.50 \(\rightarrow\) define the null hypothesis response probability: \(H_0:p=0.50\)

find total number of patients (n) and observed sample response proportion (p_hat)

summarise( n = n(), p_hat = mean(response) )

compute standard error expected under the null hypothesis

se_null = sqrt(p0 * (1 - p0) / n)

compute one-sample z statistic

z = (p_hat - p0) / se_null

compute right-tailed p-value from the standard normal distribution \(H_A:p>0.50\)

p_value = 1 - pnorm(z)

Check Assumptions

Step 6. Check model assumptions

  • checking assumptions helps validate test conditions
  • test statistic and p-value are reliable when the assumptions underlying the hypothesis test are approximately satisfied

one-sample z test depends on the following assumptions:

  • observations are independent
  • outcome is binary [response vs no response]
  • data follow a binomial model
  • sample size is large enough for the normal approximation

Calculate test statistic and determine p-value

Steps 8: Determine p-value associated with test statistic

p-value is:

  • measure of inconsistency between the hypothesized value for a population characteristic and the observed sample
  • probability of obtaining a test statistic value at least as inconsistent with as what was observed
  • quantifies the strength of evidence against \(H_0\)

In research example:

\[ p = \text{population probability of tumor response among cancer patients} \]

Visualize the p-value

Interpretation

bell-shaped curve represents the null distribution of the z statistic assuming: \(H_0: p = 0.50\)

Interpretation of regions:

  • values near the center of the curve are commonly expected if \(H_0\) is true
  • values far into the tail are less likely under \(H_0\)

Calculate and Visualize the p-value

Plot Interpretation

Blue Line: critical value for \(\alpha=0.05\) [qnorm(0.95) = 1.645]

  • standard normal distribution for a one-sided hypothesis test with: \(\alpha=0.05\)
  • identify cutoff where only 5% of the distribution lies to the right: \(P(Z \geq 1.645) = 0.05\)
  • right tailed p-value for z = 2.8: 1 - pnorm(2.8) = 0.0026
  • p-value is the tail area under the null distribution beyond the observed test statistic

State the Statistical Conclusion

Step 9: state conclusion

  • final step of hypothesis testing is making a statistical decision based on the observed evidence

Notes on comparing the observed p-value and pre-selected significance level: \(\alpha\)

p-value decision rule:

  • reject \(H_0\) if: p≤α
  • fail to reject \(H_0\) otherwise

Interpretation:

  • rejecting \(H_0\) → evidence against the null hypothesis
  • failing to reject \(H_0\) → insufficient evidence against the null hypothesis

Note: failing to reject \(H_0\) does NOT prove the null hypothesis is true, instead, it means the sample does not provide sufficiently strong evidence against it

Consider:

  • small samples may fail to detect real effects
  • high variability may weaken statistical evidence
  • absence of evidence is not the same as evidence of absence

Step 9: state conclusion

Scientific interpretation includes:

Interpretation:

  • evidence suggests that the targeted biomarker may be associated with increased tumor response probability in the population

  • among \(n = 100\) cancer patients, the observed tumor response proportion was substantially larger than what would typically be expected if the true population response probability were 50%

Summary

  • hypothesis test : an approach for evaluating whether observed sample data are consistent with a proposed population claim

  • hypothesis test can be used to help determine whether the observed data are sufficiently inconsistent with the null hypothesis to support an alternative scientific explanation

  • hypothesis testing is built on estimation and sampling variability

  • test statistic quantifies evidence relative to the null hypothesis

  • p-value measures inconsistency with the null model

  • conclusions are interpreted in scientific context
